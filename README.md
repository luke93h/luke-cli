# luke-cli
个人用脚手架,用于快速生成各种模板  
脚手架节借鉴于 [dva-cli](https://github.com/dvajs/dva-cli)
## 用法
``` javascript
    npm install luke-cli -g
    luke --demo=multiple-webpack --dest=app --no-install
```
### 参数说明
--demo 选择需要复制的代码  
--dest 输入想要输出的路径  
--no-install 复制完后是否自动npm intall，默认true  
### multiple-webpack
webpack多页应用
